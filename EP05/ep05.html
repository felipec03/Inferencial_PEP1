Pregunta a responder

Existen diferencias en el tiempo de resolucin de cubos 2x2x2 entre participantes de Uruguay, Per y Ecuador?

Solucin:

1) Leer los datos proporcionados en el CSV

datos <- read.csv2("EP05 Datos.csv")
datos_2por2 <- datos[datos$tipo == "2x2x2", ]
uruguay <- datos_2por2[datos_2por2$pais == "Uruguay", ]
peru <- datos_2por2[datos_2por2$pais == "Per", ]
ecuador <- datos_2por2[datos_2por2$pais == "Ecuador", ]

2) Familiarizacin con los datos y enunciacin de las hiptesis a contrastar

Los datos corresponden a tiempos registrados en una competencia de cubos Rubik, en este caso nos interesa nicamente la categora de 2x2x2, los grupos a docimar vendran siendo 3 paises distintos, siendo Uruguay, Per y Ecuador; nos interesa inferir acerca de las medias de mltiples muestras independientes, por lo que se proceder a enunciar las hiptesis para el posterior anlisis inferencial:

  • \(H_0\): El tiempo medio de resolucin para cubos de 2x2x2 es igual entre Uruguay, Per y Ecuador. Osease, no existe diferencia significativa entre los tiempos medios.

  • \(H_A\): El tiempo medio de resolucin para cubos de 2x2x2 es diferente para al menos uno de los 3 pases. Matemticamente, \(\exists\ i,j \in \{\text{Peru}, \text{Ecuador}, \text{Uruguay} \}, i \neq j \ |\ \mu_i \neq \mu_j\)

Para contrastar de manera eficiente, se har utilizacin del test de Anlisis de Varianza (ANOVA).

3) Anlisis de condiciones para el uso del test de ANOVA

  1. Condicin de aleatoreidad:

En este caso, el mismo ejercicio declara que la muestra fue elegida de manera aleatoria, y es posible declarar que los resultados de un participante no afecta el rendimiento o influye el tiempo de otros.

  1. Escala de variable dependiente sigue propiedad de escala de intervalos iguales:

Cmo la variable tiempo se mide en una escala de segundos hasta 2 puntos decimales de presicin, esta sigue la propiedad de intervalos iguales, por ejemplo, si pasan 4 segundos en algn tiempo medido por concursante, los mismos 4 segundos aplican para otro tiempo registrado.

  1. Suponemos que poblaciones siguen una distribucin normal
qq_peru <- ggqqplot(data = data.frame(peru$tiempo),x = "peru$tiempo",main="Grafico qq peru 2x2x2",xlab="Peruanos",ylab="tiempos")

qq_ecuador <- ggqqplot(data = data.frame(ecuador$tiempo),x = "ecuador$tiempo",main="Grafico qq ecuador 2x2x2",xlab="Ecuatorianos",ylab="tiempos")

qq_uruguay <- ggqqplot(data = data.frame(uruguay$tiempo),x = "uruguay$tiempo",main="Grafico qq Uruguayos 2x2x2",xlab="Uruguayos",ylab="tiempos")

plot_grid(qq_peru, qq_ecuador, qq_uruguay, labels = "AUTO")

  1. Prueba de homogeneidad de las varianzas:

Con las sgtes lineas de codigo averiguaremos si el conjunto de muestras que vamos a estudiar tiene la propiedad de homogeneidad:

peru_var<- var(peru$tiempo)
ecu_var <- var(ecuador$tiempo)
uru_var <- var(uruguay$tiempo)
homogeneidad = max(c(ecu_var, peru_var, uru_var)) / min(c(ecu_var, peru_var, uru_var))
print(homogeneidad)
## [1] 1.304843

Con un valor de \(\sigma ^2_{max} \div \sigma ^2_{min} = 1.3048431 \leq 1.5\) somos capaces de llevar a cabo el test ANOVA para averiguar si existen diferencias de tiempos entre los 3 pases.

4) Aplicacin de ANOVA

Cmo ya conocemos las hiptesis a contrastar y adems vemos que se cumplen las condiciones, se proceder a emplear dos test de ANOVA.

# Con prueba aov
tiempos_peru <- peru$tiempo
tiempos_ecuador <- ecuador$tiempo
tiempos_uruguay <- uruguay$tiempo

# como tienen largo distinto, se generan datos largos
# https://stackoverflow.com/questions/8320603/how-to-do-one-way-anova-in-r-with-unequal-sample-sizes
datos_largos <- data.frame(
       tiempo=c(tiempos_peru, tiempos_ecuador, tiempos_uruguay),
       pais=factor(rep(c("peru", "ecuador", "uruguay"), instancia=c(length(tiempos_peru), length(tiempos_ecuador), length(tiempos_uruguay)))))

# de la lectura 3
datos_largos[["pais"]] <- factor(datos_largos[["pais"]])
datos_largos[["instancia"]] <- factor(1:nrow(datos_largos))

prueba_aov <- aov(tiempo~pais, data = datos_largos)
summary(prueba_aov)
##              Df Sum Sq  Mean Sq F value Pr(>F)
## pais          2 0.0097 0.004844     0.3  0.742
## Residuals   114 1.8439 0.016174
p_valor_aov <- summary(prueba_aov)[[1]][["Pr(>F)"]][1]

Para la prueba empleada, usando aov(), entrega un p-value de \(0.7417617\) .

5) Anlisis Post-Hoc

Si bien no podemos rechazar la hiptesis nula, se indica en el ejercicio que se debe hacer un anlisis post-hoc, por tanto se proceder a hacer un Procedimiento HSD (honest significant differences) de Tukey.

# https://stats.stackexchange.com/questions/253588/interpreting-tukeyhsd-output-in-r
hsd <- TukeyHSD(prueba_aov, "pais", ordered = TRUE, conf.level = 0.05)
print(hsd)

En base al anlisis vemos que existe una mayor diferencia entre Ecuador y Per, puesto que posee el mayor valor en la columna diff, de igual manera, vemos que el p-valor adjunto, representado por la columna p adj es ms cercano al p-valor obtenido en el test original de ANOVA.

De todas formas, cmo no se encontr una diferencia significativa entre las 3 muestras, no tiene mucho sentido hacer el anlisis post-hoc, puesto que incluso entre Ecuador y Per no existe una diferencia suficientemente significativa.

6) Respuesta a la pregunta

Finalmente, con todo este estudio podemos concluir:

  • De la prueba de ANOVA: Cmo fue previamente mencionado, no se puede rechazar la hiptesis nula, por lo que los datos sugieren con un nivel de significancia del \(5\%\) que no existe una diferencia significativa en los tiempos medidos entre los participantes de nacionalidad Peruana, Ecuatoriana y Uruguaya.
  • Del procedimiento post-hoc: Si bien, no tiene mucho sentido emplear un procedimiento post-hoc, ya que no existe una diferencia significativa entre muestras, se concluye que entre las 3 muestras, la diferencia ms grande existe entre Ecuador y Per.
